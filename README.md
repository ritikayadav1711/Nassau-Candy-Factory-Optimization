** Factory Reallocation & Shipping Optimization Recommendation System for Nassau Candy Distributor **

Project Overview

This project develops an intelligent recommendation system for Nassau Candy Distributor that predicts shipping lead times and recommends optimal factory reallocations to improve operational efficiency and profitability.

Problem Statement

Nassau Candy currently uses fixed factory-product assignments. This can lead to:

  Higher shipping lead times
  Increased logistics costs
  Reduced profitability
  Inefficient factory utilization

The objective is to identify better factory assignments using predictive analytics and machine learning.

 Dataset

The dataset contains:

  Order Date
  Ship Date
  Product Name
  Region
  Ship Mode
  Sales
  Cost
  Gross Profit
  Units

Total Records: 10,194

Methodology

1. Data Cleaning
2. Lead Time Calculation
3. Exploratory Data Analysis (EDA)
4. Factory Mapping
5. Random Forest Regression Model
6. Recommendation Engine
7. Streamlit Dashboard Development
 Technologies Used

 Machine Learning Model

Random Forest Regressor was used to predict shipping lead times and support factory reallocation recommendations.
